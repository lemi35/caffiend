# Caffiend

Caffiend is a React application for tracking coffee consumption, caffeine intake, and spending over time.  
Users can sign up, log in, add coffee entries, and view statistics and history related to their caffeine habits.

---

## Tech Stack

- React
- Vite
- Firebase Authentication
- Firebase Firestore
- FantaCSS
- JavaScript
- CSS

---

## Features

- User authentication (sign up, login, logout)
- Per-user coffee tracking
- Coffee consumption history
- Caffeine level calculations
- Daily and total statistics
- Persistent data stored in Firebase Firestore
- Responsive UI styled with FantaCSS
